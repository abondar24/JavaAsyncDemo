# JavaAsyncDemo

Set of Java Multithreading, NIO and RX demos.

## Multithreading

Use arg "mt" to run demos.

## NIO

Use arg "nio" to run demos.

## RX

Use arg "rx" to run demos.




## Build and run
````yaml
java -jar <jar-locaiton>/async.jar <demo-type> <demo-name>
````

##Notes

- All demo names are in ()
